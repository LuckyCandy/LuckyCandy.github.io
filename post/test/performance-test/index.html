1. 性能测试关注指标解释

1.1 响应时间

对一次请求做出响应所需要的时间（2/5/8）

1.1 并发用户数

某一时间同时向软件提交请求的用户数（系统用户数 > 在线用户数 > 并发用户数） 平均并发用户数 = n(T时间内访问过系统的用户数) * L(平均操作时间)/T 预估并发用户数 = 注册用户 *(5% ~ 20%)

1.1 吞吐量

单位时间内系统处理用户的请求请求数，常用单位 请求数/秒，页面数/秒，字节/秒 吞吐量 = VU(用户数) * R(请求数) / T(测试时长) TPS
